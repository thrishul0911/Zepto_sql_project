Zepto Grocery Delivery Database Management System built using MySQL.
Implemented relational database design with Primary Keys, Foreign Keys, Joins, Aggregate Functions, GROUP BY, HAVING, Subqueries, Sorting, Filtering, 
and 15+ business analytics queries for sales, customer, and inventory analysis.
